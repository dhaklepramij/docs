# Retrieve Business Details by Business ID

This endpoint retrieves **business details** from the database based on the provided business ID.

## Path Parameters

- **id** (string, required):  
  The unique identifier of the business record to be retrieved.  
  - Example: `1234567`  
  - If a valid ID is provided, the API will return the business details.
